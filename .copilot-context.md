# 项目背景与架构规范

## 课题名称
基于Spring Boot的智能充电桩推荐系统的设计与实现

## 核心技术栈
- 后端：Java 8+, Spring Boot 2.7, MyBatis-Plus, MySQL, Redis
- 前端用户端：微信小程序原生开发
- 前端管理端：Vue 3 + Element Plus

## 系统架构
前后端分离，RESTful API，统一返回格式 `Result(code, msg, data)`。

## 核心业务模块
1. **用户端**：微信登录、附近电站推荐(协同过滤+多因子加权)、地图导航、扫码充电、微信支付。
2. **管理端**：充电桩CRUD、用户管理、订单管理、数据大屏统计。
3. **监控端**：服务器资源监控(CPU/内存)、设备异常告警。

## 核心数据库表
- user, charging_station, charging_pile, charging_order, favorite, monitor_log
